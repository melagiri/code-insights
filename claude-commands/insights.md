# Code Insights - Recent Insights

Show a summary of recent insights from my Claude Code sessions.

## Instructions

1. Run `code-insights status` to get the current sync status
2. Read the most recent insights from the local Code Insights database
3. Present insights grouped by type (decisions, learnings, work items)
4. Include the date and project for context

## Usage

- `/insights` - Show recent insights from all projects
- `/insights today` - Show insights from today only
- `/insights decisions` - Show only architectural decisions
- `/insights [project-name]` - Show insights from a specific project

## Output Format

### Recent Decisions
- Decision title (project name, date)
  Brief description

### Recent Learnings
- Learning title (project name, date)
  Brief description

### Recent Work Items
- Work item title (project name, date)
  Brief description
